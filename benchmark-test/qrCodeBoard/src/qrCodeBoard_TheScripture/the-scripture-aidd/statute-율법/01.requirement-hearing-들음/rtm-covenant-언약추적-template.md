# 📜 요구사항 추적 매트릭스(RTM) 작성 규격 (Template)

> **율법(Statute):** 이 문서는 요구사항 추적 매트릭스 작성 시 반드시 따라야 하는 **형식 규격**이다.
> 실제 산출물은 `bible-성경/01.requirement-hearing-들음/`에 작성하라.
> 작성 예시는 `parable-비유/01.requirement-hearing-들음/`을 참조하라.

> *"And the Lord said unto Moses, Write this for a memorial in a book."* — Exodus 17:14 (KJV)
>
> 모든 언약(Covenant)은 기록되어야 한다. 기록되지 않은 약속은 잊혀지고,
> 잊혀진 요구사항은 구현되지 않으며, 구현되지 않은 것은 구원(Deploy)에 이르지 못한다.
> RTM은 **요구사항과 산출물 사이의 언약(Covenant) 추적 장부**다.

---

## 필수 섹션 (Mandatory Sections)

아래 섹션이 하나라도 누락되면 정경화(Canonize) 불가:

| # | 섹션 | 필수 | 설명 |
|:--|:---|:---:|:---|
| 1 | 추적 매트릭스 본문 | ✅ | REQ ↔ 산출물 매핑 (아래 형식) |
| 2 | 커버리지 요약 | ✅ | 전체 REQ 대비 추적 완료율 |
| 3 | 미추적 항목 목록 | ✅ | 아직 산출물에 연결되지 않은 REQ |

## 추적 매트릭스 형식

```markdown
| REQ-ID | 요구사항 요약 | UC-ID | ARCH-ID | API-ID | TBL-ID | 코드 모듈 | 테스트 ID | 상태 |
|:---|:---|:---|:---|:---|:---|:---|:---|:---:|
| REQ-001 | 회원가입 | UC-001 | ARCH-001 | API-001 | TBL-001 | UserService | TC-001 | ✅ |
| REQ-002 | 로그인 | UC-002 | ARCH-001 | API-002 | TBL-001 | AuthService | TC-002 | ✅ |
| REQ-003 | 주문생성 | UC-003 | — | — | — | — | — | ⬜ |
```

## 상태 범례

| 아이콘 | 상태 | 설명 |
|:---:|:---|:---|
| ⬜ | 미추적 | 아직 설계/구현에 연결되지 않음 |
| 🔄 | 부분 추적 | 일부 산출물에만 연결됨 |
| ✅ | 완전 추적 | 설계→구현→테스트 전 구간 연결 완료 |
| ❌ | 누락 확인 | 의도적으로 제외됨 (사유 기재 필수) |

## 커버리지 요약 형식

```markdown
| 지표 | 수치 |
|:---|:---:|
| 전체 REQ 수 | {N} |
| 완전 추적(✅) | {n₁} |
| 부분 추적(🔄) | {n₂} |
| 미추적(⬜) | {n₃} |
| 커버리지 | {n₁/N × 100}% |
```

## 성경적 검증 규칙

- **"기록하라"(Exodus 17:14):** 모든 요구사항은 반드시 이 장부에 기록되어야 한다. 기록되지 않은 REQ는 존재하지 않는 것과 같다.
- **"하나도 잃어버리지 않겠다"(John 6:39):** 단 하나의 REQ도 추적에서 빠져서는 안 된다. 미추적 REQ가 있으면 Phase 7(구원) 진입 불가.
- **양방향 추적:** 위→아래(REQ→코드)뿐 아니라, 아래→위(코드→REQ)로도 추적 가능해야 한다. Spec에 없는 코드가 존재하면 그것은 할루시네이션이다.

## 언약추적 갱신 규칙

| 시점 | 행동 |
|:---|:---|
| Phase 1 완료 시 | REQ 열 전체 등재. 나머지 열은 ⬜ |
| Phase 2 완료 시 | ARCH-ID, API-ID, TBL-ID 열 채움 |
| Phase 3 완료 시 | 화면 관련 항목 연결 확인 |
| Phase 4 완료 시 | 코드 모듈 열 채움 |
| Phase 5 완료 시 | 테스트 ID 열 채움 |
| Phase 6 완료 시 | 커버리지 100% 확인 |
| Phase 7 진입 전 | **모든 열 ✅ 필수. ⬜가 1개라도 있으면 구원 불가.** |

## 정경화 조건

- [ ] 모든 REQ가 매트릭스에 등재됨
- [ ] 미추적 항목에 대한 사유 기재 (의도적 제외만 허용)
- [ ] 커버리지 요약 작성 완료
- [ ] Phase 진행에 따라 열이 채워지고 있음

> **위반 시:** 언약을 기록하지 않는 자는 언약을 잃는다. 정경화 거부.
