## 4. 인터랙션 디자인

| 이벤트 | 동작 | 피드백 |
|:---|:---|:---|
| QR 스캔 버튼 클릭 | 카메라 활성화, 뷰파인더 표시 | 로딩 스피너 → 카메라 뷰 |
| QR 인식 성공 | 결과 텍스트/URL 표시 | 성공 토스트 메시지 |
| 게시글 제목 클릭 | 상세 페이지로 이동 | hover: 밑줄 + 색상 변경 |
| 저장 버튼 클릭 | 폼 제출 → 리다이렉트 | 버튼 비활성화 + 로딩 |
| 삭제 버튼 클릭 | 확인 다이얼로그 → 삭제 | confirm 후 목록 이동 |
| 버튼 hover | 배경색 밝아짐 | transition 0.2s |

## 6. 접근성 (Accessibility)

| 항목 | 기준 | 적용 |
|:---|:---|:---|
| 색상 대비 | WCAG AA 4.5:1 | 모든 텍스트 색상 확인 완료 |
| 키보드 네비게이션 | Tab 순서 | 모든 인터랙티브 요소에 tabindex |
| 스크린리더 | aria-label | 버튼, 링크, 폼 필드에 적용 |
| 포커스 표시 | outline | 포커스 시 2px solid #6366f1 |
| 폼 레이블 | label 태그 | 모든 input에 연결된 label |
