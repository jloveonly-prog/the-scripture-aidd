## 유스케이스 상세

### UC-001: 키워드로 성경 검색

| 항목 | 내용 |
|:---|:---|
| UC-ID | UC-001 |
| 유스케이스명 | 키워드로 성경 검색 |
| 행위자 | ACT-001 (일반 사용자) |
| 연결 REQ | REQ-001, FR-001 |
| 전제 조건 | 사용자가 검색 페이지에 접근함 |
| 사후 조건 | 검색 결과가 화면에 표시됨 |

**정상 경로 (Main Path — 좁은 길):**
1. 사용자가 검색창에 키워드를 입력한다 (예: "faith")
2. 사용자가 검색 버튼을 클릭한다
3. 시스템이 KJV 66권 전체에서 키워드를 검색한다
4. 시스템이 검색 결과를 관련도 순으로 표시한다 (책명, 장:절, 본문 하이라이트)
5. 사용자가 결과 항목을 클릭하면 해당 구절의 전체 문맥이 표시된다

**대안 경로 (Alternate Path — 다른 길):**
- 3a. 사용자가 특정 책(예: Genesis)만 선택하여 검색 → 해당 책 내에서만 검색

**예외 경로 (Exception Path — 넓은 길로 빠지는 경우):**
- E1. 검색어가 빈 문자열 → "검색어를 입력하세요" 메시지 표시
- E2. 검색 결과 0건 → "결과가 없습니다. 다른 키워드를 시도하세요" 메시지 표시
- E3. 검색 시간 초과 (2초+) → "다시 시도해주세요" 에러 표시

### UC-003: 구절 북마크

| 항목 | 내용 |
|:---|:---|
| UC-ID | UC-003 |
| 유스케이스명 | 구절 북마크 |
| 행위자 | ACT-002 (회원) |
| 연결 REQ | REQ-003, FR-003 |
| 전제 조건 | 사용자가 로그인 상태 + 구절 상세 화면에 있음 |
| 사후 조건 | 해당 구절이 사용자의 북마크 목록에 저장됨 |

**정상 경로 (Main Path — 좁은 길):**
1. 사용자가 구절 상세 화면에서 북마크 아이콘을 클릭한다
2. 시스템이 해당 구절을 사용자의 북마크 목록에 저장한다
3. 시스템이 북마크 아이콘을 활성 상태로 변경한다
4. "북마크에 저장되었습니다" 토스트 메시지를 표시한다

**대안 경로:**
- 2a. 이미 북마크된 구절인 경우 → 북마크 해제 + "북마크가 해제되었습니다" 메시지

**예외 경로:**
- E1. 로그인하지 않은 상태에서 북마크 시도 → "로그인이 필요합니다" 안내 + 로그인 페이지 이동
- E2. 저장 실패 (서버 오류) → "저장에 실패했습니다. 다시 시도해주세요" 에러 표시
