# 🎨 디자인 명세서 — 성경 검색 앱 (Example)

> **비유(Parable):** 디자인 명세서가 어떻게 작성되는지 보여주는 **참조 예시**이다.

---

## 1. 디자인 시스템

### 색상 팔레트

| 용도 | 색상 | HEX |
|:---|:---|:---|
| Primary | 진한 남색 | #1a237e |
| Secondary | 금색 | #c9a84c |
| Background | 양피지 톤 | #faf6f0 |
| Text | 짙은 회색 | #2d2d2d |
| Error | 붉은색 | #c62828 |

### 타이포그래피

| 용도 | 폰트 | 크기 |
|:---|:---|:---|
| Heading | Merriweather | 28px |
| Body | Inter | 16px |
| Bible Verse | Georgia (Serif) | 18px |
