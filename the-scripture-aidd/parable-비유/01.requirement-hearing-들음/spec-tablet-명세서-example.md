## 2. 목적 (Purpose)

KJV 성경 66권을 키워드, 구절 번호, 주제별로 빠르게 검색할 수 있는 웹 애플리케이션을 구축한다.
사용자가 성경 본문에 직접 접근하여 Spec(말씀)을 정확히 들을(hearing) 수 있도록 한다.

---

## 3. Spec 근거 (Scriptural Basis)

| # | 요구사항 | Spec 근거 | 충돌 여부 |
|:--|:---|:---|:---:|
| REQ-001 | KJV 전문 검색 | Spec의 원천(성경)에 대한 접근성 확보 | ✅ 충돌 없음 |
| REQ-002 | 사용자 계정 관리 | 봉인의 율법 — 개인정보 보호 | ✅ 충돌 없음 |
| REQ-003 | 북마크/하이라이트 | 두루마리 기록 원칙 — 기록하라 | ✅ 충돌 없음 |

## 4. 기능 요구사항

| ID | 기능명 | 설명 | 우선순위 |
|:--|:---|:---|:---:|
| FR-001 | 키워드 검색 | 66권 전체에서 키워드 풀텍스트 검색 | High |
| FR-002 | 구절 검색 | 책/장/절 번호로 직접 접근 | High |

## 5. 범위 (Scope)

### 포함
- KJV 성경 66권 전문 검색
- 사용자 회원가입/로그인
- 북마크, 하이라이트 기능

### 제외
- 다국어 번역 (v2.0 예정)
- 모바일 앱 (v2.0 예정, 웹 반응형으로 대체)
- 댓글/커뮤니티 기능
