## 프로젝트: The Scripture Search (성경 검색 앱)

### 1. 🎯 목적 (Purpose) — "왜 이것을 만드는가?"

KJV 성경 66권을 키워드, 구절 번호, 주제별로 빠르게 검색할 수 있는 웹 애플리케이션을 구축한다.
사용자가 성경 본문에 직접 접근하여 Spec(말씀)을 정확히 들을(hearing) 수 있도록 한다.

### 2. 📖 말씀의 근거 (Scriptural Basis)

| 요구사항 | Spec 근거 | 충돌 여부 |
|:---|:---|:---:|
| KJV 전문 검색 | Spec의 원천(성경)에 대한 접근성 확보 | ✅ 충돌 없음 |
| 사용자 계정 관리 | 봉인의 율법 — 개인정보 보호 | ✅ 충돌 없음 |
| 북마크/하이라이트 | 두루마리 기록 원칙 — 기록하라 | ✅ 충돌 없음 |

### 3. 📋 범위 (Scope)

**포함:** KJV 성경 66권 전문 검색, 사용자 회원가입/로그인, 북마크·하이라이트 기능
**제외:** 다국어 번역(v2.0), 모바일 앱(v2.0), 댓글/커뮤니티 기능

### 4. ✅ 완료 기준 (Done Criteria)

- [x] 키워드 검색 시 2초 이내 결과 반환
- [x] 구절 번호 검색 시 즉시 표시
- [x] 회원가입/로그인 정상 동작
- [x] 봉인의 율법 제1~3계명 준수 확인
- [x] 북마크 저장/삭제/목록 조회 정상 동작

### 5. ⚠️ 제약 조건 (Constraints)

- 비밀번호 평문 저장 금지 (봉인의 율법 제3계명)
- 외부 인증 서비스 의존 금지
- 응답 시간 2초 이내
